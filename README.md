# Gym Fitness Management System

## Tech Stack
- React.js
- Node.js
- Express
- MongoDB
- JWT Authentication
- Stripe Integration

## Features
- Role-based authentication (Admin, Trainer, Member)
- Class scheduling
- Attendance tracking
- Workout & diet plans
- Subscription & payment system
- Email & SMS notifications

## Setup Instructions

### Backend
cd server
npm install
npm start

### Frontend
cd client
npm install
npm run dev
